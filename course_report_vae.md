# 基于 VAE 的艺术风格迁移与生成（实验方案草稿）

## 二、实验步骤

### 1）数据准备
- **写实域（Domain A）**：COCO 日常场景照片 10,000 张。
- **艺术域（Domain B）**：WikiArt 印象派画作 8,000 幅。
- **场景覆盖**：人物、风景、建筑、静物等类别尽量均衡，避免某类场景过度集中。
- **统一分辨率**：所有图像缩放/裁剪到 `256×256`，建议采用「先等比缩放，再中心裁剪」以减少形变。

### 2）数据预处理
- 像素归一化：将图像从 `[0,255]` 映射到 `[0,1]`。
- 数据集划分：按 `8:1:1` 划分训练/验证/测试集。
- 训练增强（可选）：随机水平翻转、轻微颜色抖动（不宜过强，避免破坏风格信息）。
- 记录并固定随机种子，保证实验可复现。

### 3）VAE 模型构建与训练（PyTorch 手动实现）

#### 网络结构
- **编码器 Encoder**：卷积下采样 + 非线性激活，输出潜在分布参数 `μ` 和 `logσ²`。
- **重参数化采样**：`z = μ + σ ⊙ ε`，其中 `ε ~ N(0, I)`。
- **解码器 Decoder**：反卷积/上采样卷积，将潜在向量恢复为 `256×256` 图像。

#### 损失函数
- 总损失：
  \[
  \mathcal{L}_{VAE} = \mathcal{L}_{rec} + \beta \cdot \mathcal{L}_{KL}
  \]
- 重建损失：MSE（或 L1，可在消融实验中对比）。
- KL 散度：
  \[
  \mathcal{L}_{KL} = -\frac{1}{2}\sum(1 + \log\sigma^2 - \mu^2 - \sigma^2)
  \]
- 建议从 `β=1.0` 起步，若出现重建模糊可尝试 `β<1`，若潜在空间分布混乱可尝试 `β>1`。

#### 训练配置
- 优化器：Adam（例如 `lr=1e-4, betas=(0.9, 0.999)`）。
- 训练轮次：50 epochs。
- 早停策略：监控验证集总损失，`patience=5~8`。
- 每个 epoch 记录训练/验证曲线（重建损失、KL 损失、总损失）。

### 4）风格迁移与生成实验
1. **写实 → 印象派**：
   - 输入写实照片 `x_real`，编码为 `z_real`，解码得到 `\hat{x}_imp`，观察印象派风格转化效果。
2. **印象派 → 写实**：
   - 输入印象派图像 `x_imp`，编码为 `z_imp`，解码得到 `\hat{x}_real`，观察写实回译效果。
3. **潜在向量融合生成**：
   - 设融合系数 `α∈[0,1]`，构造
     \[
     z_{mix}=\alpha z_{real} + (1-\alpha)z_{imp}
     \]
   - 当 `α` 从 0 到 1 递增时，生成连续过渡风格图像序列，评估平滑性与可控性。
4. **随机采样生成**：
   - 从 `N(0,I)` 随机采样 `z_rand`，经解码器生成全新风格化图像，评估多样性与可感知质量。

### 5）评估与分析

#### 定量指标
- **MSE 重建误差**：衡量像素级重建质量。
- **FID**：衡量生成图像分布与目标域分布差距（数值越低越好）。

#### 定性指标
- 邀请专业人员/具备美术背景的评审，采用 1~5 分量表评价：
  1. 风格一致性（是否接近印象派笔触、色彩和光影）
  2. 内容保持度（主体结构是否保留）
  3. 视觉自然度（是否出现明显伪影）
- 比较典型样例：成功案例、失败案例与边界样例（复杂场景、多人场景、低光照等）。

## 三、实验要求落实说明
- **避免高级封装**：仅使用 PyTorch 基础模块手动搭建 Encoder/Decoder、重参数化与损失函数。
- **数据多样性保障**：双域数据覆盖多场景并控制类别均衡。
- **多维评估**：同时给出客观指标（MSE/FID）和主观评分，确保结论完整可信。

## 四、可扩展探索（建议写入“实验改进”小节）
1. **潜在空间可视化**：
   - 使用 t-SNE / UMAP 将潜在向量降维，观察两域样本的聚类与重叠区域。
2. **融合比例敏感性分析**：
   - 系统测试 `α=0.0,0.1,...,1.0`，分析风格迁移是否线性平滑。
3. **结构消融实验**：
   - 改变卷积层深度、通道数、latent dim（如 64/128/256），对比 MSE/FID 与主观评分。
4. **训练策略对比**：
   - 对比不同 `β`（β-VAE）和不同重建损失（MSE vs L1）对“清晰度-风格化程度”的影响。

## 可直接用于报告的简短总结段
本实验以写实照片（COCO）与印象派画作（WikiArt）为双域数据，基于手动实现的 VAE 完成跨风格重建、潜在融合与随机生成。实验通过 MSE 与 FID 量化评估生成质量，并结合专家主观评分分析风格迁移效果。结果预计表明：VAE 在潜在空间连续性和可控生成方面具有优势，能够实现平滑风格过渡；但在高频细节和复杂纹理重建上仍存在一定模糊与伪影，后续可通过网络结构增强与损失函数改进进一步提升表现。
